# laba4

## 1. Move семантика в MyVector
- Конструктор по умолчанию инициализирует массив значениями от 0 до size-1
- Конструктор копирования выполняет копирование данных
- Move конструктор эффективно передает владение ресурсами
- Деструктор освобождает выделенную память
- Результаты замеров (приблизительно):
  - Копирование 10млн элементов: 0.045 сек
  - Перемещение: 0.000002 сек

## 2. Шаблонные функции
- mySwap - работает для любых типов через std::move
- containerToString - преобразует vector/list/set в строчку

## 3. Шаблонный класс MyArray
- Фиксированный размер задается в шаблоне
- Поддерживает любой тип данных
- operator[] с проверкой границ
- size() возвращает размер массива
- fill() заполняет массив указанным значением
